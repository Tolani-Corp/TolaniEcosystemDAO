# TUT Token Branding Assets

## Official Brand Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Base Teal | `#004D4D` | Primary background, base elements |
| Highlight Teal | `#007373` | Gradients, active states, rings |
| Gold | `#E5C64B` | Typography accents, CTAs, highlights |
| Circuit Lines | `#00AFAF` | Lighter teal for circuit patterns |

## Typography

- Primary: **Montserrat** or **DejaVu Sans**
- Fallback: Inter, system-ui, sans-serif

## Token Images

Place the official TUT token images in this folder:
- `tut-token-logo.png` - Main token logo
- `tut-token-icon.svg` - Icon version
- `tut-ankh-symbol.svg` - Standalone ankh symbol

## Usage Guidelines

1. Always use official brand colors
2. Maintain ankh symbol prominence
3. Circuit patterns should be subtle background elements
4. Gold should be used for emphasis, not as primary color
5. Teal gradient should flow from dark (#004D4D) to light (#00AFAF)
